# Bike Sales Dashboard (Excel)

## Project Overview
This project analyzes bike sales data using Microsoft Excel to understand customer purchasing behavior.

## Tools & Skills
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Sales Analysis
- Dashboard Creation

## Key Analysis
- Age-wise and income-wise bike purchases
- Region-wise sales performance
- Customer behavior insights
